# Kibo UI Component Catalog

Comprehensive reference of all 41+ components organized by category.

## Collaboration

### Avatar Stack
Displays multiple user avatars in an overlapping stack layout.
- Installation: `npx kibo-ui add avatar-stack`
- Use for: Team members, participant lists, collaborative features

### Cursor
Shows collaborative cursors with user names for real-time editing.
- Installation: `npx kibo-ui add cursor`
- Use for: Real-time collaboration interfaces, shared editing
